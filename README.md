# 🌐 바둑 글로벌 뉴스 대시보드

세계 바둑 뉴스·국제기전 일정·주목 선수를 한 화면에서 보는 대시보드입니다.
**파일 하나(`index.html`)로 되어 있어**, 설치나 빌드 없이 유지보수할 수 있습니다.

## 실행 (내 컴퓨터에서 확인만 할 때)
`index.html` 파일을 더블클릭하면 브라우저에서 바로 열립니다.

## 공개 주소 (모든 사람이 보는 링크)
GitHub에 올린 이 저장소가 **Vercel**에 연결되어 있어서,
아래 주소로 누구나 접속할 수 있습니다:

- 👉 (배포 후 여기에 Vercel 주소를 적어두세요. 예: https://baduk-news.vercel.app )

## 무엇을 바꾸나 (자주 하는 관리)
`index.html` 파일 맨 위 **"★★★ 설정 구역 ★★★"** 안의 세 가지만 바꾸면 됩니다.
- `SOURCES` : 뉴스 소스(RSS) 목록
- `EVENTS`  : 국제기전 일정·결과
- `PLAYERS` : 주목할 선수 이름

각 값 위에 한국어 설명 주석이 달려 있습니다.

## 바꾼 내용을 공개 주소에 반영하는 법
Claude Code에게 원하는 걸 말로 부탁하면 됩니다. 예:
> "index.html의 EVENTS에서 LG배 결과를 우승 신진서로 바꾸고, GitHub에 올려줘"

Claude Code가 파일을 고치고 GitHub에 push하면,
**Vercel이 1~2분 안에 자동으로 새 버전을 배포**합니다. (사람이 따로 올릴 필요 없음)

## 참고
- 뉴스는 `allorigins.win` 무료 중계 서버를 거쳐 가져옵니다. 가끔 이 서버가
  느리거나 잠깐 멈추면 뉴스가 안 뜰 수 있는데, 잠시 뒤 새로고침하면 됩니다.
- 'AI 요약' 버튼은 아직 "준비 중" 상태입니다(나중에 Claude API 연결 예정).

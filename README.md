# 🌐 바둑 글로벌 뉴스 대시보드

세계 바둑 뉴스·국제기전 일정·주목 선수를 한 화면에서 보는 대시보드입니다.
**파일 하나(`index.html`)로 되어 있어**, 설치나 빌드 없이 유지보수할 수 있습니다.

## 실행 (내 컴퓨터에서 확인만 할 때)
`index.html` 파일을 더블클릭하면 브라우저에서 바로 열립니다.

## 공개 주소 (모든 사람이 보는 링크)
GitHub에 올린 이 저장소가 **Vercel**에 연결되어 있어서,
아래 주소로 누구나 접속할 수 있습니다:

- 👉 (배포 후 여기에 Vercel 주소를 적어두세요. 예: https://baduk-news.vercel.app )

## 무엇을 바꾸나 (자주 하는 관리)
`index.html` 파일 맨 위 **"★★★ 설정 구역 ★★★"** 안의 세 가지만 바꾸면 됩니다.
- `SOURCES` : 뉴스 소스(RSS) 목록
- `EVENTS`  : 국제기전 일정·결과
- `PLAYERS` : 주목할 선수 이름

각 값 위에 한국어 설명 주석이 달려 있습니다.

## 바꾼 내용을 공개 주소에 반영하는 법
Claude Code에게 원하는 걸 말로 부탁하면 됩니다. 예:
> "index.html의 EVENTS에서 LG배 결과를 우승 신진서로 바꾸고, GitHub에 올려줘"

Claude Code가 파일을 고치고 GitHub에 push하면,
**Vercel이 1~2분 안에 자동으로 새 버전을 배포**합니다. (사람이 따로 올릴 필요 없음)

## 기보 다시보기 / 자동 찾기
- 오른쪽 **기보 다시보기** 목록·**기보 불러오기** 칸에서 SGF를 재생할 수 있습니다.
- 제목에 등록 선수(설정 6번 PLAYER_ALIASES)가 있는 **한국 프로대국** 기사에는 카드에
  **🔳 기보 찾기** 버튼이 붙습니다 → 한국기원에서 그 대국을 찾아 재생(정확하면 바로,
  애매하면 후보 선택). 중·일·국제 대국은 자동 소스가 없어 sgf/sgfUrl로 직접 넣습니다.
- 법적 주의: 공개·비인증 기보만, 클릭 시 1회 요청, 출처(한국기원) 표기, 전체 미러링 금지.
  서비스를 키우거나 상용화하려면 한국기원/사이버오로 등에 데이터 이용을 문의하세요.

## 구조 참고 (몰라도 됩니다)
- `index.html` : 화면 + 설정(SOURCES/EVENTS/PLAYERS/GAMES/PLAYER_ALIASES). **관리는 여기서만.**
- `api/rss.js` : 뉴스·SGF를 빠르게 가져오기 위한 Vercel 서버 함수(프록시). 건드릴 필요 없음.
- `api/kba.js` : 한국기원에서 대국 기보를 찾아주는 Vercel 서버 함수. 건드릴 필요 없음.
- 뉴스는 우리 서버 함수로 가져오고, 혹시 실패하면 공개 프록시로 자동 전환됩니다.
- 한 번 본 뉴스는 잠시 저장(캐시)해서, 다시 열 때 즉시 보여주고 뒤에서 갱신합니다.
- 'AI 요약' 버튼은 아직 "준비 중" 상태입니다(나중에 Claude API 연결 예정).
